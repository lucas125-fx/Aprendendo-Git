# Aprndendo-Git

## Comandos basicos do git

```sh
git init # para iniciar um repositório
git remote add origin link_do_repositório_do_githiub # pegar o link no git hub
git add . #adiciona todos os arquivos para serem monitorados
git commit -m ""com uma mensagem legal entre aspas"
git push origin main # recebe arquivos modificados do github
git pull origin main # recebe arquivos modificados ou excluidos
# mostra se tem arquivos criados, modificados ou excluidos.
# tambem informa so o repo remoto esté atualizado com o local e vice-versa
git status
```
